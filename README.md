Este proyecto trata de instalar el VS CODE y git en mi pc para después configurar el GIT e instalar unas extensiones en el VS Code para distintas tareas
## Diseño de la aplicación
TaskFlow es una app de gestión de tareas que permite:
-Añadir tareeas
-Marcar las completadas
-Eliminar tareas
-Ver estadísticas básicas
