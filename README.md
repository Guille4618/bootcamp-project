# TaskFlow

Aplicación web de gestión de tareas desarrollada con HTML, 
CSS, JavaScript y Tailwind CSS.

## Funcionalidades

- Añadir tareas con prioridad (alta, media, baja)
- Marcar tareas como completadas
- Eliminar tareas con confirmación
- Sistema de dependencias entre tareas
- Filtrar tareas por estado
- Búsqueda de tareas en tiempo real
- Estadísticas con barra de progreso
- Modo oscuro
- Datos persistentes con LocalStorage

## Tecnologías usadas

- HTML5 semántico
- Tailwind CSS via CDN
- JavaScript vanilla
- LocalStorage para persistencia
- Vercel para el despliegue

## Cómo ejecutarlo

1. Clona el repositorio:
```
git clone https://github.com/Guille4618/bootcamp-project.git
```
2. Abre `index.html` en tu navegador
3. O visita la versión desplegada en: [URL de Vercel]

## Ejemplos de uso

- Añade una tarea escribiendo en el campo de texto y seleccionando la prioridad
- Haz doble clic sobre una tarea para editarla
- Usa los filtros para ver solo las tareas pendientes o completadas
- Crea dependencias entre tareas para bloquear unas hasta que otras estén completadas
